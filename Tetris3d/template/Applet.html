Rules:

As in simple tetris game, the goal is to fit the block into the net so that they occupy all the space on the bottom of the cuboid. As soon as any level is full, it will disappear, leaving space for new blocks. The game is over if the blocks reach the top of the cuboid.

Operating the camera's view:

You can change the perspective by pressing q, a, e, d keys

You can zoom in / zoom out by pressing w, s keys

Moving the block around:

You can move the block around using the arrow keys

You can rotate the block using the x, c and f, v keys

You can speed up the block pressing space bar
